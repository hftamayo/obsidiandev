
==instalacion del proyecto

```
pnpm create vite vanillatstodo --template vanilla-ts
cd vanillatstodo
pnpm install
pnpm run dev

Instalar Tailwind:
pnpm add tailwindcss postcss autoprefixer
npx tailwindcss init -p


creando el mismo proyecto pero con yarn:
yarn create vite vanillatstodo --template vanilla-ts
yarn install
yarn add tailwindcss postcss autoprefixer
npx tailwindcss init -p
cp postcss.config.js, tailwind.config.js to .
cp src/index.css ./src

```
